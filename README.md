# ストップウォッチゲーム（Processing）
【Processing】ランダムな目標タイムに合わせて時間を計測し、平均・誤差を算出するストップウォッチゲームのソースコードです。
![ストップウォッチゲーム](https://user-images.githubusercontent.com/84171334/188277036-8aaedf98-6d07-450c-8f55-4d986df8194e.jpg)

## 実行内容
- 目標タイムをランダムに設定（3~6秒）
- 計測タイムを記録（3回）
- 平均（下3桁）
- 誤差（下7桁）
